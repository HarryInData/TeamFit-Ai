How it works

Three steps. Zero guesswork.

01
Students join the room

A session code gets everyone into a shared workspace — JWT-secured, role-separated, no browser extension required.

02

Teams write and run code in the Monaco editor. TeamFit AI logs contributions in the background — students see nothing different.

Session ends → AI synthesizes data → professor receives a per-student score, summary, and full activity breakdown. Grade with confidence.
